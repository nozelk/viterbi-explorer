05 · Scenariji

Pripravljeni primeri

Hitro naloži parametre v demo in se igraj.

2 stanj · 2 simbolov · T=4

Vreme → Dežnik →

Klasičen uvodni primer. Skrita stanja so vreme (sončno/deževno), opazovanja pa ali soseda opaziš z dežnikom. Iz zaporedja dežnikov ugibamo vreme.

Stanja: Soncno, Dezevno
Zaporedje: BrezD. → Dezn. → Dezn. → BrezD.
2 stanj · 3 simbolov · T=3

Sladoled (Jurafsky) →

Primer iz knjige Jurafsky & Martin. Skrita stanja: HOT/COLD dan. Opazovanja: stevilo pojedenih sladoledov (1, 2 ali 3).

Stanja: HOT, COLD
Zaporedje: 3 → 1 → 3
2 stanj · 3 simbolov · T=5

Razpoloženje → Objave →

Skrita stanja so razpolozenje osebe (veselo/zalostno), opazovanja pa tip objave na socialnem omrezju.

Stanja: Veselo, Zalostno
Zaporedje: Meme → Citat → Citat → Selfie → Meme
2 stanj · 3 simbolov · T=6

Borza → Tržni režim →

Klasična uporaba HMM v financah (regime-switching model). Skrita stanja so tržni režim: Bikov trg (rast) ali Medvedji trg (padec). Opazovanja so dnevni premiki indeksa: Gor, Dol, ali Mir. Iz zaporedja dnevnih gibov skušamo ugotoviti, kakšno je bilo ozadje trga.

Stanja: Bikov, Medvedji
Zaporedje: Gor → Gor → Dol → Dol → Dol → Gor
